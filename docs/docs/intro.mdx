# Introduction

**Flipova Foundation** is a unified design system and UI toolkit for building consistent experiences across **React Native** (iOS, Android) and **React Web**.

It ships as a single npm package with multiple sub-entries — each tree-shakeable and independently importable.

## What's inside

| Sub-module | What it gives you |
|---|---|
| `@flipova/foundation` | React Native primitives + components + layouts (main entry) |
| `@flipova/foundation/web` | Pure DOM equivalents — no React Native dependency |
| `@flipova/foundation/tokens` | Static design tokens (spacing, colors, radii, typography…) |
| `@flipova/foundation/theme` | ThemeProvider, useTheme, 9 built-in themes |
| `@flipova/foundation/config` | defineConfig, FoundationProvider, token overrides |
| `@flipova/foundation/layout` | Registry, hooks, RN layout components |

## Architecture at a glance

```
@flipova/foundation
├── tokens/          Static values: spacing, colors, radii, shadows, typography, motion
├── theme/           React context: ThemeProvider, useTheme, 9 themes, useColorScheme
├── config/          FoundationProvider, defineConfig, resolveConfig
├── layout/
│   ├── registry/    Declarative metadata used by Flipova Studio
│   ├── hooks/       useBreakpoint, usePlatformInfo, useSafeArea
│   └── ui/
│       ├── primitives/  Box, Stack, Inline, Center, Scroll, Divider (RN)
│       └── components/  Button, TextInput, Badge, Avatar… (RN)
└── web/             Pure DOM equivalents of every primitive and component
```

## Two deployment targets

### React Native apps

```tsx
import { Box, Button, Stack } from '@flipova/foundation';
import { FoundationProvider, defineConfig } from '@flipova/foundation/config';

export default function App() {
  return (
    <FoundationProvider config={defineConfig({ defaultTheme: 'light' })}>
      <Stack spacing={4}>
        <Box p={4} bg="card" borderRadius="lg">
          <Button variant="primary" label="Hello Foundation" />
        </Box>
      </Stack>
    </FoundationProvider>
  );
}
```

### React Web apps (Vite, Next.js…)

```tsx
import { Box, Button, Stack, ThemeProvider } from '@flipova/foundation/web';

export default function App() {
  return (
    <ThemeProvider defaultTheme="light">
      <Stack spacing={4}>
        <Box p={4} borderRadius="lg">
          <Button variant="primary">Hello Foundation Web</Button>
        </Box>
      </Stack>
    </ThemeProvider>
  );
}
```

:::tip Web entry point
The `/web` sub-module is built with `platform: "browser"` — esbuild resolves `useColorScheme.web.ts` instead of the React Native version, so `react-native` is never bundled into your web app.
:::

## Flipova Studio

Foundation also bundles **Flipova Studio** — a local visual builder that generates React Native code:

```bash
npx flipova-studio
# → starts the visual builder at http://localhost:4200
```

See the [Studio guide](./guides/studio) for full documentation.

## Installation

```bash
npm install @flipova/foundation
```

For React Native peer deps:

```bash
npx expo install \
  expo-linear-gradient expo-haptics expo-status-bar \
  react-native-gesture-handler react-native-reanimated \
  react-native-safe-area-context react-native-screens \
  lucide-react-native
```
