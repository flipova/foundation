# API Reference

Welcome to the Flipova Foundation API reference documentation. This section provides detailed information about all components, utilities, and types available in the library.

## Core Modules

### Design Tokens
Access the design system tokens for consistent styling across your application.

- **Colors** - Color palette and semantic color tokens
- **Spacing** - Spacing scale and utilities
- **Typography** - Typography scale and font utilities
- **Borders** - Border radius and border width tokens
- **Shadows** - Shadow depth tokens

### Components
Pre-built React Native components ready to use in your application.

- **ThemeProvider** - Theme context provider
- **Button** - Button component with variants
- **Card** - Card container component
- **FlexLayout** - Flexbox layout component
- **BentoLayout** - Bento grid layout component

### Layout Primitives
Advanced layout components for complex UI structures.

- **GridLayout** - Grid-based layout
- **DeckLayout** - Card deck layout
- **SwiperLayout** - Swipeable card layout
- **MasonryLayout** - Masonry grid layout
- **ParallaxLayout** - Parallax scrolling layout
- **CrossTabLayout** - Cross-tab navigation layout

### Utilities
Helper functions and utilities for common tasks.

- **useTheme** - Hook for accessing theme context
- **applyDefaults** - Apply default props to components
- **createStyles** - Create styled components with theme support

### Types
TypeScript type definitions for type safety.

- **Theme** - Theme interface definition
- **DesignTokens** - Design tokens type definitions
- **ComponentProps** - Component prop type definitions

## Usage Example

```tsx
import { ThemeProvider, Button, Card, useTheme } from '@flipova/foundation';
import { tokens } from '@flipova/foundation';

function MyComponent() {
  const { theme } = useTheme();
  
  return (
    <ThemeProvider>
      <Card style={{ padding: tokens.spacing.lg }}>
        <Button variant="primary" onPress={() => {}}>
          Click me
        </Button>
      </Card>
    </ThemeProvider>
  );
}
```

## Type Safety

All components and utilities are fully typed with TypeScript. You can leverage autocomplete and type checking in your IDE:

```tsx
import type { ButtonProps, Theme } from '@flipova/foundation';

const buttonProps: ButtonProps = {
  variant: 'primary', // Type-safe: only 'primary' | 'secondary' | 'ghost'
  size: 'medium',    // Type-safe: only 'small' | 'medium' | 'large'
};
```
