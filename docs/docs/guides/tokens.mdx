# Design Tokens

Design tokens are the **static, platform-agnostic values** that underpin every component in Foundation. They live in `@flipova/foundation/tokens` and are never runtime-computed.

```ts
import { spacing, colors, radii, shadows, fontSizes, fontWeights } from '@flipova/foundation/tokens';
```

---

## Spacing

All spacing tokens are **pixel values** (numbers). Use them as `p`, `m`, `gap`, etc. on primitives.

```ts
const spacing = {
  0:  0,
  1:  4,
  2:  8,
  3:  12,
  4:  16,
  5:  20,
  6:  24,
  7:  28,
  8:  32,
  10: 40,
  12: 48,
  14: 56,
  16: 64,
  20: 80,
  24: 96,
  28: 112,
  32: 128,
  36: 144,
  40: 160,
  44: 176,
  48: 192,
  52: 208,
  56: 224,
  60: 240,
  64: 256,
} as const;

type SpacingToken = keyof typeof spacing; // 0 | 1 | 2 | 3 | 4 …
```

### Usage

```tsx
// on Box / Stack / Inline primitives:
<Box p={4} m={2} gap={3} />                 // tokens: 16, 8, 12 px

// direct value:
import { spacing } from '@flipova/foundation/tokens';
const styles = { padding: spacing[4] };      // 16
```

## Border radius

```ts
const radii = {
  none: 0,
  xs:   2,
  sm:   4,
  base: 6,
  md:   8,
  lg:   12,
  xl:   16,
  '2xl':20,
  '3xl':24,
  full: 9999,
} as const;

type RadiusToken = keyof typeof radii;
```

### Usage

```tsx
<Box borderRadius="lg" />          // 12 px
<Button borderRadius="full" />     // pill shape
```

## Colors

Colors are static **hex strings** grouped into a flat palette plus named sub-objects for seasonal themes.

```ts
import { colors } from '@flipova/foundation/tokens';

colors.primary500   // '#000091'
colors.gray200      // '#E5E7EB'
colors.success500   // '#22C55E'
colors.neon.primary // '#9200db'
```

### Semantic aliases (from `useTheme()`)

| Token | Meaning |
|---|---|
| `theme.primary` | Brand primary color |
| `theme.background` | Page background |
| `theme.card` | Surface / card background |
| `theme.foreground` | Main text |
| `theme.muted` | Subtle background |
| `theme.mutedForeground` | Subtle / secondary text |
| `theme.border` | Dividers and outlines |
| `theme.destructive` | Danger / error actions |
| `theme.success` | Positive feedback |
| `theme.warning` | Caution |
| `theme.info` | Informational |

## Shadows

Shadows are React Native `StyleSheet`-compatible objects (not CSS strings).

```ts
const shadows = {
  none: { shadowColor:'#000', shadowOffset:{width:0,height:0}, shadowOpacity:0, shadowRadius:0, elevation:0 },
  sm:   { shadowColor:'#000', shadowOffset:{width:0,height:1}, shadowOpacity:0.05, shadowRadius:2, elevation:2 },
  base: { shadowColor:'#000', shadowOffset:{width:0,height:2}, shadowOpacity:0.10, shadowRadius:4, elevation:3 },
  md:   { shadowColor:'#000', shadowOffset:{width:0,height:4}, shadowOpacity:0.15, shadowRadius:8, elevation:5 },
  lg:   { shadowColor:'#000', shadowOffset:{width:0,height:8}, shadowOpacity:0.20, shadowRadius:16, elevation:8 },
  xl:   { shadowColor:'#000', shadowOffset:{width:0,height:12}, shadowOpacity:0.25, shadowRadius:24, elevation:12 },
  '2xl':{ shadowColor:'#000', shadowOffset:{width:0,height:16}, shadowOpacity:0.30, shadowRadius:32, elevation:16 },
} as const;

type ShadowToken = keyof typeof shadows;
```

### Usage on Box

```tsx
<Box shadow="md" />   // applies elevation + shadowColor + shadowOpacity etc.
```

## Overriding tokens at app level

Static tokens can be overridden per-app using `defineConfig`. The overrides are merged into the resolved config and made available via `useTokens()`:

```ts
import { defineConfig } from '@flipova/foundation/config';

export default defineConfig({
  tokens: {
    spacing: { 0:0, 1:4, 2:8, 3:12, 4:16, 5:20, 6:24, 8:32 },
    radii:   { none:0, sm:4, base:6, md:8, lg:12, xl:16, '2xl':20, full:9999 },
  },
});
```

```tsx
import { useTokens } from '@flipova/foundation/config';

function MyComponent() {
  const { spacing, radii } = useTokens();
  return <div style={{ padding: spacing[4], borderRadius: radii.md }} />;
}
```

:::note
`defineConfig` overrides are available through `useTokens()`. The raw static tokens in `@flipova/foundation/tokens` are never mutated.
:::
