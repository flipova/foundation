# Components

All components are **theme-aware** — they read colors from the active `ThemeProvider` via `useTheme()` internally. No explicit color props needed unless you want to override.

Components are available in two flavours:

| Import | Platform |
|---|---|
| `@flipova/foundation` | React Native (View, Text, StyleSheet…) |
| `@flipova/foundation/web` | React DOM (div, span, input…) |

The props API is **identical** across both entry points.

---

## Button

```tsx
import { Button } from '@flipova/foundation/web';

<Button variant="primary" size="md" onClick={() => {}}>
  Save changes
</Button>
```

### Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `variant` | `'primary' \| 'secondary' \| 'outline' \| 'ghost' \| 'destructive'` | `'primary'` | Visual style |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Height and padding |
| `label` | `string` | — | Text label (alternative to children) |
| `icon` | `ReactNode` | — | Icon node |
| `iconPosition` | `'left' \| 'right'` | `'left'` | Icon placement |
| `loading` | `boolean` | `false` | Shows spinner, disables interaction |
| `disabled` | `boolean` | `false` | Disables the button |
| `fullWidth` | `boolean` | `false` | Stretches to full container width |
| `borderRadius` | `RadiusToken` | `'md'` | Override border radius |
| `onPress` | `() => void` | — | Native press handler (RN) |
| `onClick` | `MouseEventHandler` | — | Web click handler |

### Variants

```tsx
<Button variant="primary">     Primary action</Button>
<Button variant="secondary">   Secondary action</Button>
<Button variant="outline">     Outlined</Button>
<Button variant="ghost">       Ghost / subtle</Button>
<Button variant="destructive"> Delete / danger</Button>
```

## TextInput / Input

```tsx
import { TextInput } from '@flipova/foundation/web';

<TextInput
  label="Email address"
  placeholder="you@example.com"
  variant="outlined"
/>
```

### Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `label` | `string` | — | Label rendered above the input |
| `error` | `string` | — | Error message below the input |
| `variant` | `'outlined' \| 'filled' \| 'underline'` | `'outlined'` | Visual style |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Height |
| `secureEntry` | `boolean` | `false` | Password field |
| `borderRadius` | `RadiusToken` | `'md'` | Override radius |

## Avatar


### Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `source` | `string` | — | Image URL |
| `initials` | `string` | — | 1-2 letter fallback |
| `variant` | `'circle' \| 'square'` | `'circle'` | Shape |
| `size` | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` | Diameter (24/32/40/56/72 px) |
| `background` | `string` | `theme.muted` | Fallback bg color |

## Pattern — Form

```tsx
import { Stack, TextInput, TextArea, Button, Checkbox } from '@flipova/foundation/web';

function ContactForm() {
  return (
    <Stack spacing={4} style={{ maxWidth: 480 }}>
      <TextInput label="Full name" placeholder="Jane Doe" />
      <TextInput label="Email"     placeholder="jane@example.com" />
      <TextArea  label="Message"   numberOfLines={4} />
      <Checkbox  label="Subscribe to newsletter" />
      <Button variant="primary" fullWidth>Send message</Button>
    </Stack>
  );
}
```

## Accessibility notes

- Every interactive component sets appropriate `role`, `aria-*` and keyboard handlers.
- `Button` uses a native `<button>` element on web — fully keyboard and screen-reader accessible.
- `TextInput` / `Checkbox` / `Switch` are associated with their `<label>` via `htmlFor`.
- Minimum touch target size: **40 × 40 px** (`size="md"`), **48 × 48 px** (`size="lg"`).
