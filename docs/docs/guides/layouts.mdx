# Layouts & Primitives

Foundation provides two levels of layout abstraction:

1. **Primitives** — low-level building blocks (`Box`, `Stack`, `Inline`, `Center`, `Scroll`, `Divider`)
2. **Layout components** — higher-level containers for common app patterns (`DashboardLayout`, `AuthLayout`, etc.)

---

## Primitives

### Box

The single lowest-level primitive. Everything is a `Box`.

```tsx
import { Box } from '@flipova/foundation/web';

<Box
  p={4}
  m={2}
  bg="card"
  borderRadius="lg"
  shadow="base"
  flexDirection="row"
  alignItems="center"
  gap={3}
  width="100%"
>
  children
</Box>
```

#### Spacing props

All spacing props accept a `SpacingToken` (numeric key: 0, 1, 2, 3, 4…).

| Prop | CSS equivalent |
|---|---|
| `p` | padding |
| `pt / pr / pb / pl` | padding-top / right / bottom / left |
| `px` | padding-left + right |
| `py` | padding-top + bottom |
| `m` | margin |
| `mt / mr / mb / ml` | margin-top / right / bottom / left |
| `mx` | margin-left + right |
| `my` | margin-top + bottom |
| `gap` | gap |
| `rowGap / columnGap` | row-gap / column-gap |

#### Style props

| Prop | Type |
|---|---|
| `bg` | string (any CSS color or theme key) |
| `gradient` | theme gradient key or `[color1, color2, ...]` |
| `gradientDirection` | `'toTop' \| 'toBottom' \| 'toLeft' \| 'toRight' \| …` |
| `borderRadius` | `RadiusToken` |
| `shadow` | `ShadowToken` |
| `flex` | number |
| `flexDirection` | `'row' \| 'row-reverse' \| 'column' \| 'column-reverse'` |
| `alignItems` | `'stretch' \| 'flex-start' \| 'flex-end' \| 'center' \| 'baseline'` |
| `justifyContent` | `'flex-start' \| 'flex-end' \| 'center' \| 'space-between' \| …` |
| `width / height` | CSS size string or number |
| `minWidth / maxWidth / minHeight / maxHeight` | CSS size |
| `position` | CSS position |
| `top / right / bottom / left` | CSS offset |
| `zIndex` | number |
| `overflow` | CSS overflow |
| `opacity` | 0–1 |
| `as` | HTML tag for polymorphism (web only) |
| `theme` | override the local theme mode |

---

### Stack

Vertical flex column with uniform spacing.

```tsx
import { Stack } from '@flipova/foundation/web';

<Stack spacing={4} align="center" distribute>
  <Header />
  <Content />
  <Footer />
</Stack>
```

| Prop | Type | Default | Description |
|---|---|---|---|
| `spacing` | `SpacingToken` | `0` | Gap between children |
| `align` | `alignItems value` | `'stretch'` | Cross-axis alignment |
| `justify` | `justifyContent value` | `'flex-start'` | Main-axis alignment |
| `reverse` | `boolean` | `false` | `column-reverse` |
| `distribute` | `boolean` | `false` | `space-between` |
| `fillHeight` | `boolean` | `false` | `flex: 1` |

---

### Inline

Horizontal flex row with uniform spacing.

```tsx
import { Inline } from '@flipova/foundation/web';

<Inline spacing={3} align="center" wrap>
  <Avatar />
  <Badge label="Admin" />
  <Text>John Doe</Text>
</Inline>
```

| Prop | Type | Default | Description |
|---|---|---|---|
| `spacing` | `SpacingToken` | `0` | Gap between children |
| `align` | `alignItems value` | `'center'` | Vertical alignment |
| `justify` | `justifyContent value` | `'flex-start'` | Horizontal alignment |
| `wrap` | `boolean` | `false` | Allow wrapping |
| `reverse` | `boolean` | `false` | `row-reverse` |

---

### Center

Centers content horizontally and/or vertically.

```tsx
import { Center } from '@flipova/foundation/web';

<Center style={{ height: 200 }}>
  <Spinner />
</Center>

<Center horizontal>     {/* horizontal only */}
<Center vertical>       {/* vertical only (needs height) */}
```

---

### Scroll

Scrollable container.

```tsx
import { Scroll } from '@flipova/foundation/web';

<Scroll direction="vertical" style={{ maxHeight: 400 }}>
  {items.map(i => <Row key={i.id} item={i} />)}
</Scroll>
```

| Prop | Type | Default |
|---|---|---|
| `direction` | `'vertical' \| 'horizontal' \| 'both'` | `'vertical'` |
| `hideScrollbar` | `boolean` | `false` |

---

### Divider

Thin separator line.

```tsx
import { Divider } from '@flipova/foundation/web';

<Divider />
<Divider orientation="vertical" style={{ height: 24 }} />
```

## Layout components

Higher-level containers for common app patterns. Available from both `@flipova/foundation` and `@flipova/foundation/web`.

---

### RootLayout

Full-screen root wrapper. Sets background, safe-area padding, and scroll.

```tsx
<RootLayout background={theme.background} scrollable padding={4}>
  <AppContent />
</RootLayout>
```

---

### CenteredLayout

Centers a constrained content block, optionally inside a card.

```tsx
<CenteredLayout maxWidth={480} padding={5} shadowed borderRadius="xl">
  <LoginForm />
</CenteredLayout>
```

---

### AuthLayout

Split-panel auth screen: branding panel (desktop) + form panel.

```tsx
<AuthLayout
  branding={<BrandPanel />}
  brandingBackground={theme.primary}
  brandingRatio={0.45}
  formMaxWidth={420}
>
  <LoginForm />
</AuthLayout>
```

---

### DashboardLayout

Classic admin layout: fixed header, collapsible sidebar, scrollable content, optional footer.

```tsx
<DashboardLayout
  header={<Topbar />}
  sidebar={<Sidebar />}
  footer={<StatusBar />}
  sidebarWidth={256}
  defaultSidebarCollapsed={false}
  disableContentScroll={false}
>
  <PageContent />
</DashboardLayout>
```

---

### SidebarLayout

Simpler layout: content with a side panel, collapsible on mobile.

```tsx
<SidebarLayout
  sidebar={<FilterPanel />}
  sidebarWidth={280}
  position="left"
  collapsible
>
  <ResultsList />
</SidebarLayout>
```

---

### SplitLayout

Two-pane layout with a separator — email client style.

```tsx
<SplitLayout
  left={<MessageList />}
  right={<MessageDetail />}
  leftWidth={320}
  orientation="horizontal"
  hideLeftOnMobile
/>
```

---

### GridLayout

Responsive CSS grid with configurable columns, gap, and optional item background.

```tsx
<GridLayout columns={3} spacing={4} maxWidth={1200}>
  {products.map(p => <ProductCard key={p.id} product={p} />)}
</GridLayout>
```

---

### FlexLayout

Flex container with direction, wrap, alignment, and optional max-width.

```tsx
<FlexLayout direction="row" wrap spacing={3} justify="space-between">
  <Chip label="React Native" />
  <Chip label="TypeScript" />
  <Chip label="Expo" />
</FlexLayout>
```

---

### FooterLayout

Keeps footer pinned at the bottom of the viewport.

```tsx
<FooterLayout
  footer={<BottomNav />}
  footerHeight={64}
  sticky
>
  <Content />
</FooterLayout>
```

---

### ResponsiveLayout

Adaptive layout with three modes: `basic`, `sidebar`, `full` (header + sidebar + content + footer).

```tsx
<ResponsiveLayout
  adaptiveMode="full"
  header={<AppHeader />}
  sidebar={<AppSidebar />}
  content={<AppContent />}
  footer={<AppFooter />}
  sidebarWidth={240}
  headerHeight={60}
/>
```

## Pattern — App shell

```tsx
import {
  DashboardLayout,
  Stack,
  Inline,
  Box,
  Button,
  Avatar,
  Text,
} from '@flipova/foundation/web';

function Topbar() {
  return (
    <Inline spacing={3} alignItems="center" justify="space-between" style={{ padding: '0 24px', height: 60 }}>
      <Text fontSize={18} fontWeight="700">MyApp</Text>
      <Inline spacing={2}>
        <Button variant="ghost" size="sm">Settings</Button>
        <Avatar initials="JD" size="sm" />
      </Inline>
    </Inline>
  );
}

function Sidebar() {
  return (
    <Stack spacing={1} p={2}>
      <Button variant="ghost" fullWidth>Dashboard</Button>
      <Button variant="ghost" fullWidth>Users</Button>
      <Button variant="ghost" fullWidth>Analytics</Button>
    </Stack>
  );
}

export default function App() {
  return (
    <DashboardLayout header={<Topbar />} sidebar={<Sidebar />} sidebarWidth={220}>
      <Box p={5}>
        <Text fontSize={24} fontWeight="700">Dashboard</Text>
      </Box>
    </DashboardLayout>
  );
}
```

## useBreakpoint

Responds to the active breakpoint:

```tsx
import { useBreakpoint } from '@flipova/foundation/layout';

function Responsive() {
  const bp = useBreakpoint(); // 'xs' | 'sm' | 'md' | 'lg' | 'xl' | '2xl'

  if (bp === 'xs' || bp === 'sm') return <MobileView />;
  return <DesktopView />;
}
```
